Welcome to the website's readme file! This document provides an overview of the website and its features, along with instructions for getting started.

About the Website
SD Bank is a bank teller UI. It provides basic transactions that a teller can do, making it a quick solution for tellers. With a user-friendly interface and a range of useful features, SD Bank is the ideal solution for small banks.

Features
Search Customer
Open an account
Deposit / withdraw money
Getting Started
Prerequisites
To use SD Bank, you will need:

A modern web browser, such as Google Chrome, Mozilla Firefox, or Safari.
A stable internet connection.
Installation
SD Bank is a web-based application, so there is no software to install. Simply navigate to the website's URL in your web browser to get started.

Login
To access all of SD Bank's features, you will need to create an account. To do so, click on the "Sign Up" button on the homepage, and follow the on-screen instructions.

If you already have an account, click on the "Log In" button and enter your email address and password to access your account.
